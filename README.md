<h1>Calculator UI Project: Self-Check</h1>

<p>Simple Calculator UI designed using concepts from CPSC 5330 Module 3.</p>

<ul>
<li>Compiles: Project builds with no errors.</li>
<li>Use of StackViews:</li>
  <ul>
<li>Root vertical StackView contains display and buttons stacks.</li>
<li>Display StackView has 2 labels (expression + result).</li>
<li>Buttons StackView has 5 horizontal rows, each with 4 buttons.</li>
<li>Responsive: Works on different screen sizes (iPhone and iPad).</li>
<li>Orientation: Works in both portrait and landscape.</li>
      </ul>
<li>UI designed with colors and spacing for readability; buttons fill space evenly.</li>
</ul>
